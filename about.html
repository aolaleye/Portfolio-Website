About Me

I am a designer and developer who is passionate about user research, clean design, and organized code.

My passion for web development stemmed from two of my favorite interests - design and technology. Soon after taking my first course in web development, I came to the understanding that the two disciplines often overlap. As a developer, I revel in the ability to explore elements of creativity through technology.

My portfolio showcases my work on several different passion projects. In order to cultivate rich web experiences for users, I practice mobile-first, responsive web design, and I use client-side frameworks to resolve design problems, create smart user interfaces and implement provoking interactive elements. I also like to stay up-to-date on advances in web development, the latest trends in web design and the newest features in technology.

If you would like to delve a little deeper into my code, feel free to check out my GitHub.

Tools

HTML

Solid understanding of MDN web standards, as well as organizing and manipulating the DOM. Able to develop HTML content with clean and structured code. Familiar with HTML5 elements, input elements and select menus. Able to style web pages and navigation bars. Comfortable adding pages to websites. Can structure and organize HTML tables using table elements, such as radio buttons, fieldsets and legends. Familiar with the HTML video and audio elements. Able to caption video and implement custom media players.

CSS

Able to utilize CSS to create mobile-first responsive designs. Solid grasp of CSS3 basic selectors, values and units. Extensive practice with CSS transitions and transforms, as well as with animating SVGs. Familiar with the box model and floating elements. Comfortable using media queries and breakpoints to trigger CSS display modes and to position page content. Strong understanding of flexbox properties and growing and aligning flex items. Strong knowledge of CSS3 attribute selectors, advanced pseudo-classes, pseudo-elements and combinators.

Also proficient with CSS frameworks such as Bootstrap and Materialize.

JavaScript

Able to store and track information with variables, arrays, two-dimensional arrays and objects. Strong knowledge of conditional statements and boolean values. Can create reusable code with functions and simplify repetitive tasks with loops. Able to use arrays to track multiple items and objects to track data. Aim to consistently practice "DRY" Programming with the goal of reusing code in the future. Able to formulate AJAX requests with JavaScript and jQuery. Solid grasp on constructor functions and prototypes.

jQuery

Strong understanding of the features and utility functions of jQuery. Comfortable with researching jQuery documentation. Aim to utilize jQuery to traverse the DOM without inhibiting the goal of graceful degradation. Able to roll out and implement custom jQuery plugins. Familiar with input validation techniques and approaches. Solid knowledge of browser rendering behavior and performance and proficient in identifying cross-browser compatibility issues and ways to work around such issues. Good grasp on asynchronous request handling, partial page updates and creating AJAX requests with jQuery.

Bootstrap

Solid understanding of the Bootstrap 4 documentation. Able to use the Bootstrap grid to create responsive layouts and to solve common layout, typography and browser compatibility problems. Able to implement Bootstrap navbars, buttons, dropdown menu components and modals into websites. Comfortable adding custom CSS to websites built with Bootstrap.

Git

Able to commit and stage changes in Git repositories and comfortable using GitHub to increase positive workflow performance. Working knowledge of pushing and pulling, branching and merging conflicts, and working with remote repositories.
